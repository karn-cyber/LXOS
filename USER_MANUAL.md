## Events Management

### Creating an Event

1. Go to **Events** → Click **Create Event**
2. Fill in event details:
   - **Title**: Event name
   - **Description**: What the event is about
   - **Type**: Club, Clan, or LX event
   - **Dates**: Start and end date/time
   - **Budget**: Allocated budget
   - **Attendees**: Expected number

3. **Select Organization**:
   - For Club events: Choose the club
   - For Clan events: Choose the clan
   - For LX events: No selection needed

4. **Add Requirements** (optional):
   - Item name
   - Quantity needed
   - Estimated cost
   - Click **Add Requirement**

5. **Book a Room** (optional):
   - Select from available rooms
   - System checks for conflicts automatically

6. Click **Create Event**

### Event Status

- **PENDING**: Awaiting approval
- **APPROVED**: Approved and scheduled
- **REJECTED**: Not approved (see reason)
- **COMPLETED**: Event has finished

### Editing Events

1. Go to event detail page
2. Click **Edit Event** (if you have permission)
3. Update details
4. Click **Save Changes**

**Note:** Approved events can only be edited by admins.

### Approving Events (Admin/LX Team)

1. Go to event detail page
2. Review all details
3. Click **Approve** or **Reject**
4. If rejecting, provide a reason
5. Confirm action

## Budget & Expenses

### Viewing Budget Dashboard (Admin/Finance)

1. Go to **Budget**
2. View overall statistics
3. See club and clan budget breakdowns
4. Monitor expense categories

### Creating an Expense

1. Go to **Expenses** → Click **Create Expense**
2. Fill in details:
   - **Event**: Link to related event
   - **Title**: Expense name
   - **Amount**: Cost in ₹
   - **Category**: Food, Venue, Equipment, etc.
   - **Description**: Additional details
   - **Paid Date**: When payment was made

3. Click **Submit**

### Verifying Expenses (Finance/Admin)

1. Go to **Expenses**
2. Find pending expense
3. Click to view details
4. Click **Approve** or **Reject**
5. Confirm action

**Important:** Approving an expense automatically updates the event's budget!

## Achievements

### Recording an Achievement

1. Go to **Achievements** → Click **Add Achievement**
2. Fill in details:
   - **Title**: Achievement name
   - **Description**: What was achieved
   - **Category**: Competition, Award, etc.
   - **Date**: When it happened
   - **Points**: Points to award to clan (optional)

3. **Link to Event/Club/Clan** (optional):
   - Select related event
   - Select club
   - Select clan

4. **Add Participants**:
   - Enter participant names
   - Click **Add** for each
   - Remove with trash icon

5. Click **Create Achievement**

**Important:** Points are automatically added to the clan leaderboard!

## File Repository

### Uploading Files

1. Go to **Files** → Click **Upload File**
2. Select file from computer
3. Fill in metadata:
   - **Description**: What the file contains
   - **Tags**: Keywords for searching
   - **Event/Club**: Link to related items
   - **Semester**: Academic period

4. Click **Upload**

### Finding Files

1. Go to **Files**
2. Use filters:
   - By event
   - By club
   - By tag
3. Click file to view details
4. Click **Download** to get file

## Tips & Best Practices

### Events

✅ **DO:**
- Book rooms early to avoid conflicts
- Provide detailed descriptions
- Add all requirements upfront
- Submit for approval well in advance

❌ **DON'T:**
- Create duplicate events
- Skip required fields
- Exceed allocated budget

### Expenses

✅ **DO:**
- Link expenses to events
- Keep receipts/bills
- Submit promptly after payment
- Categorize correctly

❌ **DON'T:**
- Submit without documentation
- Exceed event budget
- Duplicate expense entries

### Budget

✅ **DO:**
- Monitor budget regularly
- Plan expenses in advance
- Track spending vs. allocation

❌ **DON'T:**
- Overspend allocated budget
- Ignore budget warnings

## Troubleshooting

### Can't Create Event

**Issue:** "Forbidden" error
- **Solution:** Check if you have permission (Admin, LX Team, or Club Head)

### Room Conflict

**Issue:** "Room not available"
- **Solution:** Choose different time or room

### Budget Exceeded

**Issue:** Can't add expense
- **Solution:** Request budget increase from admin

### Login Issues

**Issue:** Can't sign in
- **Solution:** Verify email and password, contact admin if needed

## Glossary

- **Approval**: Process of reviewing and accepting/rejecting requests
- **Budget Allocated**: Total budget assigned
- **Budget Spent**: Amount already used
- **Budget Remaining**: Allocated minus spent
- **CRUD**: Create, Read, Update, Delete operations
- **Event Type**: CLUB, CLAN, or LX
- **Status**: Current state (PENDING, APPROVED, etc.)
